> Client-side application for the port project.

## Install

```bash
go install github.com/eslusarenko/port-client@latest
```

Or download a pre-built binary from [Releases](https://github.com/eslusarenko/port-client/releases).

## Usage

```bash
port --help
```

## Build from source

```bash
cd client
make build
```
